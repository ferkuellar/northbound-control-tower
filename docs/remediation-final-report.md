# Remediation Final Report

The final report is generated after post-remediation validation.

## Structure

1. Executive Summary
2. Request Metadata
3. Original Finding
4. Remediation Template
5. Terraform Plan Summary
6. Security Scan Summary
7. Cost Estimate Summary
8. Approval Summary
9. Apply Summary
10. Post-Validation Result
11. Evidence List
12. Final Decision
13. Recommendations

## Audiences

Technical reviewers use the JSON and evidence list. Executives use the Markdown summary and final decision.

## Evidence Chain

The report links plan, gates, approval, apply, outputs, rescan, validation result, and diff artifacts so the remediation can be audited end to end.
